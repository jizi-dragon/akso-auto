## 核心流程
```
创建 → 填写字段 → 保存 → 提交 → 待接收确认状态
```

## 操作步骤

### Step 1: 导航到模块
- 菜单路径：质量管理 → MAH 质量信息通知 2.0 → [子菜单名称待补充]
- 操作：展开左侧菜单树，找到对应模块入口
- 预期页面：MAH 质量信息通知列表页

### Step 2: 创建新工单
- 目标元素：页面上的"新建"/"创建"按钮
- 操作：click
- 预期结果：弹出创建表单或跳转到创建页面

### Step 3: 填写表单字段
- 遍历表单字段，根据字段类型选择合适的填写方式
- **富文本字段**：使用键盘逐字输入，确保数据绑定生效
- **下拉框**：选择后重新 snapshot 确认选项已更新
- **日期选择**：按标准日期格式输入
- 按表单自上而下顺序填写

### Step 4: 保存
- 目标元素："保存"按钮
- 操作：click
- 预期结果：提示"保存成功"，工单进入草稿状态

### Step 5: 提交
- 目标元素："提交"按钮
- 操作：click
- 预期结果：工单状态变为"待接收确认"

## 配置规则
- 规则1：所有必填字段必须完整填写才能保存
- 规则2：保存后可修改，提交后不可修改（需撤回）
- 规则3：提交后工单进入"待接收确认"状态
